# RG Framework

Генератор отчетов с графическим интерфейсом на PyQt.

## Возможности
- 📝 ГОСТ-титульник
- ⌨️ Автозаполнение по Tab
- 📚 Иерархия разделов и подразделов
- 📊 Таблицы, формулы, списки
- 🚀 Массовая генерация (8 лаб за 2 минуты)
- 🔧 Горячие клавиши (Alt+1, Alt+2, Ctrl+S...)

## Установка
```bash
pip install -r requirements.txt
python main.py
